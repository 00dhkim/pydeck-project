# pydeck-project

1. deck.gl의 python binding인 pydeck 라이브러리의 사용법을 익힘
2. 출발지와 목적지만 있을 때 중간 경로를 알려주는 osrm API의 사용법을 익힘
3. 대구광역시 나드리콜 접수 데이터를 전처리
4. 전처리된 데이터로 trips layer를 생성하여 마무리
